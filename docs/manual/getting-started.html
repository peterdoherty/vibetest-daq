Prerequisites

RequirementMinimum versionWhere to get it
Python3.10python.org or your system package manager
NI-DAQmx driverLatestNI Package Manager (Windows) or ni.com
NI cDAQ-9174 + NI 9230 modules—Connected via USB and visible in NI MAX
Important: The NI-DAQmx driver must be installed before you install the Python package. Without the driver, the nidaqmx Python library cannot communicate with the hardware.

Step 1 — Verify Hardware in NI MAX

Open NI Measurement & Automation Explorer (NI MAX) and expand Devices and Interfaces. You should see your cDAQ chassis and both NI 9230 modules listed. Note the device names — they will look like cDAQ1Mod1 and cDAQ1Mod2. You will need these names in the application settings.

Tip: If the modules do not appear, try a different USB port, check the cable, and make sure the NI-DAQmx driver is installed.

Step 2 — Create a Virtual Environment

It is good practice to install Python packages in a virtual environment rather than system-wide. Open a terminal and run:

python -m venv .venv

Activate the environment:

PlatformCommand
Windows (PowerShell).venv\Scripts\Activate.ps1
Windows (cmd).venv\Scripts\activate.bat
macOS / Linuxsource .venv/bin/activate

Step 3 — Install the Package

From the root of the vibetest-daq repository, install in editable mode so that changes to the source take effect immediately:

python -m pip install -e .

This installs three main dependencies automatically:

  • nidaqmx — Python bindings for the NI-DAQmx C driver
  • numpy — numerical array library used for data handling
  • PySide6 — Qt 6 bindings for the graphical interface

If you plan to run the linter or tests, install the development extras too:

python -m pip install -e ".[dev]"

Step 4 — Launch the Application

Start the GUI:

vibetest-daq-gui

Or run a command-line acquisition:

vibetest-daq --output vibration_data --rate 5000

If the GUI window appears and shows "Ready", the installation is complete.

Step 5 — First Acquisition

  1. In the GUI, confirm that Module 1 device and Module 2 device match the names you saw in NI MAX.
  2. Choose an output directory using the Browse button.
  3. Leave the sample rate at 5000 Hz and block duration at 10 s for a quick test.
  4. Press Start Acquisition.
  5. Watch the level meters respond to vibration. After a few blocks, press Stop.
  6. Open the output directory and inspect the CSV files.
